# dummy-words

## Installation

Using npm:

```
npm install dummy-words
```

Using yarn

```
yarn install dummy-words
```

## Example

In (ES6)

```
import {all, random} from 'dummy-words';
```

In CommonJS

```
const dummyNames = require('dummy-words');
```

To Generate random names

```
    random()
```

To Get all names at once

```
    all
```
